# Telegram Mini App для Тахмины

Готовый стартовый проект на **Vite + Vanilla JavaScript**.

## Что внутри
- полноэкранный видео-фон
- приветствие для Тахмины
- автоматическая текущая дата
- отдельные приветствия на **1 апреля**
- печать текста по буквам
- сообщения по тапу на экран
- блокировка тапа, пока текст печатается
- исчезновение текста через 5 секунд
- готово для GitHub и Railway

## Как запустить

```bash
npm install
npm run dev
```

## Как собрать

```bash
npm run build
npm run start
```

## Куда положить видео

Положи своё видео сюда:

```bash
public/video/space-loop.mp4
```

Сейчас в проекте лежит файл-заглушка `public/video/README.txt`.

## Деплой
Подходит для Railway.
